# sao-qms

> The starter template for QMS system

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao qms my-project
```

### From git

```bash
sao kirill zaidulin/sao-qms my-project
```
